
# Projet Sport

Application web de gestion d'activités sportives et d'abonnements.

## Structure du projet

```
app/
├── controllers/        # Contrôleurs (logique métier)
├── core/              # Classes de base (Router, Model, Controller)
├── models/            # Modèles de données
├── views/             # Templates HTML
│   ├── abonnements/
│   ├── activites/
│   ├── auth/
│   ├── communication/
│   ├── home/
│   └── layout/
└── img/               # Images
```

## Fonctionnalités

- **Authentification** : inscription, connexion, récupération mot de passe
- **Activités** : boxe, lutte, MMA, musculation, yoga
- **Abonnements** : gestion des abonnements utilisateurs
- **Communication** : contact, à propos

## Installation

1. Placer le dossier dans `xampp/htdocs/Promo321/info/projet_sport/`
2. Configurer la base de données
3. Accéder via `http://localhost/Promo321/info/projet_sport/`

## Technologies

- PHP (MVC)
- MySQL
- HTML/CSS/JavaScript
